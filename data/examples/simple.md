# Урок 1: Введение в программирование

## Цели урока
- Понять, что такое программирование
- Познакомиться с базовыми понятиями: алгоритм, программа, язык программирования
- Написать первую программу "Hello, World!"

## Ключевые понятия

### Что такое программирование?
Программирование — это процесс создания инструкций (алгоритмов), которые выполняет компьютер.

### Алгоритм
Последовательность шагов для решения задачи.

### Язык программирования
Средство для записи алгоритмов в понятном компьютеру виде.
